# TaUTus

A all-in-one development tool for Python Ubuntu Touch Applications

## Features

- Dependency free
- Create Clickable Apps
- Add Python libraries to your apps
- Some shipped in default files to make your life easier

## How to get started

0. Install Python 3
1. Get an instance of TaUTus
2. Initialise a new project with the command `init`.  
   The full command would look like this:  
   `./tautus.pyz init <directory name>`
3. Complete the initalisation
4. Change directory into the new project
5. Do something with the new project
6. Run on desktop with `./tautus.pyz build desktop`
